# CLAUDE.md

Guidance for working in this repository.

## Repository conventions

- **Do not commit ephemeral, feature-building docs.** Plans, specs, design
  scratch, and other working notes generated while building a feature are
  throwaway — keep them local and out of git history. `/docs/superpowers/` is
  git-ignored for exactly this reason. The only docs that belong in the repo are
  durable ones aimed at someone setting up or using the project (e.g.
  `README.md`).
